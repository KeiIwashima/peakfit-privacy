# PeakFit プライバシーポリシー

最終更新日: 2026年5月11日

## 1. 基本方針

PeakFit(以下「本アプリ」)は、株式会社 clout(以下「当社」)が提供する筋力トレーニング・栄養管理アプリです。当社は、ユーザーの個人情報を適切に保護することの重要性を認識し、本ポリシーに従って情報を取り扱います。

## 2. 取得する情報

本アプリは以下の情報を取得します:

### 2-1. ユーザー入力情報
- プロフィール情報(名前、性別、生年月日、身長、体重、体脂肪率)
- トレーニング記録(種目、重量、回数、セット数)
- 食事記録(食材、量、カロリー、栄養素)
- 体組成データ(体重推移、体脂肪率推移)
- 目標情報(コンテスト日、目標体重など)

### 2-2. HealthKit データ(ユーザー許可後のみ)
- 体重
- 体脂肪率
- 歩数
- アクティブカロリー

## 3. 情報の利用目的

取得した情報は以下の目的のためにのみ使用されます:
- トレーニング進捗の追跡
- カロリーおよび栄養素の計算
- 目標達成のための分析
- ピーク週設定とズレ検知機能の提供

## 4. 情報の保存場所

すべての情報は、ユーザーの iOS デバイス内のローカルデータベースに保存されます。当社のサーバーやクラウドサービスにはデータを送信しません。

## 5. 第三者への提供

当社は、ユーザーの情報を第三者に提供することはありません。

## 6. データの削除

ユーザーは、アプリ内の設定画面からいつでも全データを削除することができます。また、iOS の設定 → 一般 → iPhone ストレージ → PeakFit → App を削除 により、すべてのデータを完全に削除できます。

## 7. HealthKit データの取り扱い

HealthKit から取得したデータは、本アプリのカロリー計算と進捗分析にのみ使用されます。HealthKit データを広告、マーケティング、または当社外への提供のために使用することはありません。
